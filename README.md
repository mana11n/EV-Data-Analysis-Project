## Overview
This project presents an interactive Tableau dashboard built to analyze electric vehicle (EV) population data across the United States. The analysis focuses on adoption trends over time, vehicle type distribution (BEV vs PHEV), manufacturer dominance, geographic spread by state, and Clean Alternative Fuel Vehicle (CAFV) eligibility.

The goal of this project is to explore large-scale structured data and communicate insights effectively through interactive visualizations.

---

## Dataset
- **Dataset Name:** Electric Vehicle Population Data  
- **Publisher:** Washington State Department of Licensing  
- **Data Portal:** Washington State Open Data  
- **Format:** CSV  

The dataset contains information such as:
- Vehicle type (BEV / PHEV)
- Model year
- Manufacturer and model
- State-wise registration counts
- Electric range
- CAFV eligibility status

> Note: The dataset file is not included in this repository due to GitHub file size limitations. The data can be accessed from the official Washington State open data portal by searching for **‚ÄúElectric Vehicle Population Data‚Äù**.

## Key Insights
- Electric vehicle adoption shows a strong upward trend in recent model years
- Battery Electric Vehicles (BEVs) account for the majority of total EV registrations
- EV adoption is geographically concentrated in specific states
- Tesla leads overall vehicle counts among manufacturers
- CAFV eligibility highlights the policy-driven aspect of EV adoption
