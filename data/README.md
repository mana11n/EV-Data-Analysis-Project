# Dataset Information

This project uses the **Electric Vehicle Population Data** dataset.

## Source
- Publisher: Washington State Department of Licensing
- Portal: Washington State Open Data
- Dataset Name: Electric Vehicle Population Data

## Description
The dataset contains information related to electric vehicle registrations, including:
- Vehicle type (BEV / PHEV)
- Model year
- Manufacturer and model
- Electric range
- Clean Alternative Fuel Vehicle (CAFV) eligibility
- Geographic distribution

## Availability
The dataset file is not included in this repository due to GitHub file size limitations.

To access the dataset, visit the Washington State Open Data portal and search for:
**“Electric Vehicle Population Data”**
